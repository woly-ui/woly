tooltip

import { Tooltip } from "woly";

Tooltip is used when you want to show addition information near component. Tooltip message is visible when user hover over component that is passed as a children to Tooltip.

Example

Подсказка может быть написана в две строки и более
Подсказка
Подсказка может быть написана в две строки
Подсказка
<Playground>
  <block.S style={{ display: 'flex', padding: '100px' }}>
    <Tooltip message="Подсказка может быть написана в две строки и более" position="bottom">
      <Button
        onClick={() => console.info('Hi!')}
        text="Button"
        priority="primary"
        style={{ marginRight: '10px' }}
      />
    </Tooltip>
    <Tooltip message="Подсказка" position="top" id="tooltip-top">
      <Button
        onClick={() => console.info('Hi!')}
        text="Button"
        priority="primary"
        style={{ marginRight: '10px' }}
      />
    </Tooltip>
    <Tooltip message="Подсказка может быть написана в две строки" position="left">
      <Button
        onClick={() => console.info('Hi!')}
        text="Button"
        priority="primary"
        style={{ marginRight: '10px' }}
      />
    </Tooltip>
    <Tooltip message="Подсказка" position="right" priority="primary">
      <Input
        name="name"
        onChange={() => console.info('On input change')}
        type="text"
        priority="primary"
      />
    </Tooltip>
  </block.S>
</Playground>
