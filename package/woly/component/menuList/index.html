menuList

Use it

import { menuList } from "woly";

Installation

npm install woly
# or
yarn add woly

menuList is a list of items related to a single object. This component can be used in menu dropdown or input autosuggestion components.

Example

  • Red
  • Green
  • Blue
const Example = () => (
  <menuList.Base
    menu={[
      { id: 0, item: 'Red' },
      { id: 1, item: 'Green' },
      { id: 2, item: 'Blue' },
    ]}
  />
);
