checkbox

Use it

import { checkbox } from "woly";

Installation

npm install woly
# or
yarn add woly

checkbox is used to turn on/off an option. Checkbox can be provided with label using label component.

Example

Travel packing checklist:
passport
money
tickets
const Example = () => (
  <checkbox.Base
    label={<label.Primary text="Checkbox label" />}
    isChecked={true}
    onChange={(e) => console.info('change', e)}
    id="checkbox"
  />
);
