checkbox

Use it

import { checkbox } from "woly";

Installation

npm install woly
# or
yarn add woly

Checkbox is used to turn on/off an option.

Using checkbox

Checkbox can be provided with label using label component.

Installation

npm install woly
# or
yarn add woly

Usage

import { checkbox, label } from 'woly';
const Example = () => (
  <checkbox.Base
    label={<label.Primary text="Checkbox label" />}
    isChecked={isChecked}
    onChange={onChange}
    id="checkbox"
  />
);
