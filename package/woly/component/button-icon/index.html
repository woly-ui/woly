button-icon

Use it

import { buttonIcon } from "woly";

Installation

npm install woly
# or
yarn add woly

buttonIcon is a button with an icon

Example

<buttonIcon.Base
  icon={<Icon />}
  onClick={(event) => console.info('buttonIcon clicked')}
/>
