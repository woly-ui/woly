select

import { Select } from "woly";

Select component to choose single option from list of options.

Example

Use Select to choose one option from dropdown. You can pass placeholder if you don't have initial selected value. Each option has data attribute data-value where value of option is saved.

Choose an option
  • Toyota
  • Opel
  • BMW
<Playground size="S">
  <div style={{ height: '150px' }}>
    <StateEvent initial={null} change={(event) => event?.target?.dataset.value}>
      {(value, change) => (
        <Select
          icon={<ArrowDown />}
          onChange={change}
          options={[
            { children: 'Toyota', value: 'Toyota' },
            { children: 'Opel', value: 'Opel' },
            { children: 'BMW', value: 'BMW' },
          ]}
          placeholder="Choose an option"
          selected={value}
        />
      )}
    </StateEvent>
  </div>
</Playground>

Selected value

Or you can pass initial selected value.

Cats

BMW
  • Toyota

<Playground size="S">
  <div style={{ height: '150px' }}>
    <StateEvent initial="Cats" change={(event) => event?.target?.dataset.value}>
      {(value, change) => (
        <div style={{ display: 'flex' }}>
          <Select
            icon={<ArrowDown />}
            onChange={change}
            options={[
              { children: 'Cats', value: 'Cats' },
              { children: 'Dogs', value: 'Dogs' },
              { children: 'Birds', value: 'Birds' },
            ]}
            selected={value}
            variant="secondary"
          />
          <div style={{ marginLeft: '10px' }}>
            <Select
              icon={<ArrowDown />}
              onChange={() => {}}
              options={[
                { children: 'Toyota', value: 'Toyota' },
                { children: 'Opel', value: 'Opel' },
                { children: 'BMW', value: 'BMW' },
              ]}
              selected="BMW"
              isDisabled
            />
          </div>
        </div>
      )}
    </StateEvent>
  </div>
</Playground>
