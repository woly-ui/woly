chip

Use it

import { chip } from "woly";

Installation

npm install woly
# or
yarn add woly

Chip is used for categorizing or markup. Chips can be used inside input or as tags.

Example

const Example = () => (
  <Chip
    icon={<CloseIcon />}
    onClick={(event) => console.info('Chip clicked')}
    text="Chip"
  />
);
