inputPassword

Use it

import { inputPassword } from "woly";

Installation

npm install woly
# or
yarn add woly

InputPassword is used for a secure password entry. The component has a button-icon to show and hide user password.

Using inputPassword

InputPassword can be used in an authorization form or to confirm user action.

Installation

npm install woly
# or
yarn add woly

Usage

import { inputPassword } from 'woly';
const Example = () => (
  <inputPassword.Base
    name="input"
    iconHidden={<IconHidden />}
    iconOpen={<IconOpen />}
    value={value}
  />
);
