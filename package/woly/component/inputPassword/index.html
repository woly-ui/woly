inputPassword

Use it

import { inputPassword } from "woly";

Installation

npm install woly
# or
yarn add woly

inputPassword is used for a secure password entry. The component has a button-icon to show and hide user password. InputPassword can be used in an authorization form or to confirm user action.

Example

const Example = () => (
  <inputPassword.Base
    onChange={(e) => console.info('change', e)}
    name="input"
    iconHidden={<OpenEyeIcon />}
    iconOpen={<ClosedEyeIcon />}
    value="Just a value"
  />
);
