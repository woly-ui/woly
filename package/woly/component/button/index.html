button

import { Button } from "woly";

Button allows users to take actions and make choices with a single tap.

Example

<Playground>
  <Button
    text="Just a button"
    variant="primary"
    left={<SearchIcon />}
    onClick={(event) => alert('Hi!')}
  />
</Playground>

Variants

Primary and secondary variants are should be used to focus user attention.

<Playground>
  <Button text="The main action on the page" variant="primary" />
  <Button text="Alternative action" variant="secondary" />
</Playground>

Icons

Button can be used with icon on the left side or the right. Use icons on the both sides is not recommended.

<Playground>
  <Button text="Icon on the left" variant="primary" left={<ArrowLeft />} />
  <Button text="Icon on the right" variant="primary" right={<SearchIcon />} />
  <Button
    text="Wow! Both"
    variant="primary"
    right={<SearchIcon />}
    left={<ArrowLeft />}
  />
</Playground>

Sizes

Size controlled by the component-level block property not from the props.

<Playground>
  <block.S>
    <Button variant="primary" text="Small" />
  </block.S>
  <block.M>
    <Button variant="primary" text="Small" />
  </block.M>
  <block.XL>
    <Button variant="primary" text="Huge" />
  </block.XL>
</Playground>
