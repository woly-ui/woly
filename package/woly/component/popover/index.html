popover

Use it

import { popover } from "woly";

Installation

npm install woly
# or
yarn add woly

Popover is a modal window.

Popover is used when you want to show the content in the same window. It creates the floating layer over the current page to get user feedback or display information. You can use Popover with Surface component.

Example

Popover is open
const Example = () => <Popover isOpen={true}>Popover content</Popover>;
