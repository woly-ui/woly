button-floating

Use it

import { buttonFloating } from "woly";

Installation

npm install woly
# or
yarn add woly

buttonFloating is placed at the bottom right side of the screen and contains an icon

Example

{' '}

const Example = () => (
  <buttonFloating.Base
    icon={Icon}
    onClick={(event) => console.info('buttonFloating clicked')}
  />
);
