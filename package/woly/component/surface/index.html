surface

Use it

import { surface } from "woly";

Installation

npm install woly
# or
yarn add woly

Surface is a simple layer that contains information. Surface can be used inside Popover or Sidebar.

Example

Surface content
const Example = () => <surface.Base>Surface content</surface.Base>;

Kinds

NameDescription
BaseBase surface

Props

Pass any content as a children property within Surface component.
