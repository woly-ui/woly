`Checkbox` is used to turn on/off an option. Checkbox can be provided with label using `Label` component.

### Example

<CheckboxExample />

```tsx
const Example = () => (
  <Checkbox
    icon={<Icon />}
    id="checkbox"
    isChecked={true}
    onChange={(e) => console.info('change', e)}
    text="Checkbox text"
    variant="base"
  />
);
```

### Kinds

| Name       | Description                                                 |
| ---------- | ----------------------------------------------------------- |
| `Checkbox` | Base checkbox. Useful for creating a new kind of checkboxes |

### Props

| Name        | Type                                         | Default     | Description                                         |
| ----------- | -------------------------------------------- | ----------- | --------------------------------------------------- |
| `icon`      | `React.ReactNode`                            |             | Set the icon component inside the checkbox          |
| `id`        | `string`                                     |             | HTML id attribute                                   |
| `isChecked` | `boolean`                                    |             | Whether checkbox is checked or not                  |
| `text`      | `string`                                     |             | Text component                                      |
| `onChange`  | `React.ChangeEventHandler<HTMLInputElement>` |             | Callback when checkbox is clicked                   |
| `variant`   | `string`                                     | `'default'` | Variant prop to style Checkbox component            |
