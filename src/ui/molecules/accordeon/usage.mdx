The accordion component delivers large amounts of content in a small space through progressive disclosure.
The header title give the user a high level overview of the content allowing the user to decide which sections to read.

Accordions can make information processing.
However, it does hide content from users and itâ€™s important to account for a user not noticing or reading all of the included content.

As ussuall accordions are used to organize related information.
In another case, they shold be used when space is at a premium and long content cannot be displayed all at once,
eg in a side panel.

### Kinds

| Name        | Description                                                 |
| ----------- | ----------------------------------------------------------- |
| `Accordion` | Base accordion. Useful for creating a new kind of accordion |

### Props

| Name       | Type              | Default     | Description                               |
| ---------- | ----------------- | ----------- | ----------------------------------------- |
| `children` | `React.ReactNode` |             | Trigger element of accordion              |
| `content`  | `React.ReactNode` |             | Content of accordion                      |
| `isOpen`   | `boolean`         | `'false'`   | Whether the component is visible or not   |
| `variant`  | `string`          | `'default'` | Variant prop to style accordion component |
