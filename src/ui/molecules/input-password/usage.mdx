`InputPassword` is used for a secure password entry.
The component has a button-icon to show and hide user password. InputPassword can be used in an authorization form or to confirm user action.

### Example

<InputPasswordExample />

```tsx
const Example = () => (
  <InputPassword
    onChange={(e) => console.info('change', e)}
    name="input"
    iconHidden={<OpenedEyeIcon />}
    iconOpen={<ClosedEyeIcon />}
    value="Just a value"
  />
);
```

### Kinds

| Name            | Description                                                         |
| --------------- | ------------------------------------------------------------------- |
| `InputPassword` | Base inputPassword. Useful for creating a new kind of inputPassword |

### Props

| Name         | Type                                         | Default     | Description                                   |
| ------------ | -------------------------------------------- | ----------- | --------------------------------------------- |
| `disabled`   | `boolean`                                    | `null`      | HTML disabled attribute                       |
| `iconHidden` | `React.ReactNode`                            | `null`      | Replace base icon to hide password            |
| `iconOpen`   | `React.ReactNode`                            | `null`      | Replace base icon to show password            |
| `name`       | `string`                                     |             | HTML name attribute                           |
| `onChange`   | `React.ChangeEventHandler<HTMLInputElement>` |             | Callback when input is changed                |
| `value`      | `HTMLInputElement['value']`                  | `null`      | HTML value attribute                          |
| `variant`    | `string`                                     | `'default'` | Variant prop to style InputPassword component |
