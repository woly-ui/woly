`Toggle` is used to switch between two different options. Toggle can be provided with label using `Label` component.

### Example

<ToggleExample />

```tsx
const Example = () => (
  <Toggle
    label={<Label text="Toggle label" />}
    isChecked={false}
    onChange={(e) => console.log('change', e)}
    id="toggle"
  />
);
```

### Kinds

| Name     | Description                                           |
| -------- | ----------------------------------------------------- |
| `Toggle` | Base toggle. Useful for creating a new kind of toggle |

### Props

| Name        | Type                                         | Default     | Description                                       |
| ----------- | -------------------------------------------- | ----------- | ------------------------------------------------- |
| `id`        | `string`                                     |             | HTML id attribute                                 |
| `isChecked` | `boolean`                                    |             | Whether toggle is checked or not                  |
| `label`     | `React.ReactNode`                            | `null`      | Label component                                   |
| `onChange`  | `React.ChangeEventHandler<HTMLInputElement>` |             | Callback when toggle is clicked                   |
| `variant`   | `string`                                     | `'default'` | Variant prop to style Toggle component            |
| `...`       | `HTMLInputElement`                           | `{}`        | Other props are inherited from `HTMLInputElement` |
