Popover is based on Surface and opens on click on element or custom logic.

For example, Select can be created on InputContainer, Popover and List. Also, DateSelect can be created with Calendar, InputContainer and Popover.

### Kinds

| Name      | Description  |
| --------- | ------------ |
| `Popover` | Base popover |

### Props

| Name      | Type      | Default     | Description                             |
| --------- | --------- | ----------- | --------------------------------------- |
| `isOpen`  | `boolean` |             | Whether the component is visible or not |
| `variant` | `string`  | `'default'` | Variant prop to style Popover component |
| `...`     | `object`  | `{}`        | Other props passed to the component     |
