`TextArea` defines a multi-line text input control. TextArea is often used in a form, to collect user inputs like comments or reviews.

### Example

<TextAreaExample />

```tsx
const Example = () => (
  <TextArea
    name="name"
    placeholder="Enter some text here"
    value="Just a value"
    onChange={(event) => console.info('On text change')}
  />
);
```

### Kinds

| Name       | Description                                                 |
| ---------- | ------------------------------------------------------------|
| `TextArea` | Base text-area. Useful for creating a new kind of text-area |

### Props

| Name       | Type                                                          | Default     | Description                                       |
| ---------- | ------------------------------------------------------------- | ----------- | ------------------------------------------------- |
| `name`     | `string`                                                      |             | Name attribute specifies a name of the text-area  |
| `disabled` | `boolean`                                                     | `null`      | HTML disabled attribute                           |
| `value`    | `HTMLInputElement['value']`                                   | `null`      | Value of text-area field                          |
| `onChange` | `(e: React.ChangeEvent<HTMLInputElement>) => unknown`         |             | Callback when user text-area                      |
| `onKeyDown`| `(event: React.KeyboardEvent<HTMLTextAreaElement>) => unknown`|             | Callback when user text-area                      |
| `variant`  | `string`                                                      | `'default'` | Variant prop to style TextArea component          |
| `...`      | `HTMLInputElement`                                            | `{}`        | Other props are inherited from `HTMLInputElement` |
