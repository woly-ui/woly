import { Label } from './index';

`Label` is a description text for located near interactive component. Label can be passed as a property to a Checkbox or a Toggle component.

### Example

<Label text="Simple label" />

```tsx
const Example = () => <Label text="Simple label" />;
```

### Kinds

| Name    | Description                                         |
| ------- | --------------------------------------------------- |
| `Label` | Base label. Useful for creating a new kind of label |

### Props

| Name      | Type               | Default     | Description                                       |
| --------- | ------------------ | ----------- | ------------------------------------------------- |
| `text`    | `React.ReactNode`  |             | Label text                                        |
| `variant` | `string`           | `'default'` | Variant prop to style Input component             |
| `...`     | `HTMLLabelElement` | `{}`        | Other props are inherited from `HTMLLabelElement` |
