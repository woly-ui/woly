`Tooltip` is used to show additional information near the component. The information should be contextual, useful, and nonessential.

A tooltip can be a simple message box that is displayed when a user hovers over, clicks, or gives focus to a UI element
such as an icon, a highlighted word, or a button that is passed as children to `Tooltip`.
Also, the definition tooltip provides additional help or defines an item or term. It may be used on the label of a UI element,
or on a word embedded in a paragraph.

Interactive tooltips may contain rich text and other interactive elements like buttons or links.
In general, hiding interactive content in a tooltip is discouraged.
Interactive tooltips are best used for onboarding experiences and product tours.

The tooltip position is flexible and changes depending on how close the element is to the edge of the screen.
The top, right, bottom, and left positions appear outside the target element, center-aligned along the appropriate axis.

### Example

    <Tooltip

    >

    </Tooltip>
    <Tooltip text={<Text type="S">Подсказка</Text>} position="top" id="tooltip-top">
      <Button onClick={() => console.info('Hi!')} text="Button" variant="primary" />
    </Tooltip>
    <Tooltip
      column
      text={
        <Text type="S">
          Подсказка может быть написана в две и более строки, а также содержать блок с actions
        </Text>
      }
      position="left"
      iconRight={
        <>
          <Button onClick={() => console.info('Hi!')} text="Назад" variant="primary" />
          <Button onClick={() => console.info('Hi!')} text="Вперед" variant="secondary" />
        </>
      }
    >
      <Button onClick={() => console.info('Hi!')} text="Button" variant="primary" />
    </Tooltip>
    <Tooltip
      iconLeft={<IconInfo />}
      text={<Text type="XS">Подсказка c иконкой</Text>}
      position="right"
      variant="primary"
    >
      <Input
        name="name"
        onChange={() => console.info('On input change')}
        type="text"
        variant="primary"
      />
    </Tooltip>
  </block.S>

### Props

| Name        | Type                 | Default       | Description                                                                              |
| ----------- | -------------------- | ------------- | ---------------------------------------------------------------------------------------- |
| `children`  | `React.ReactElement` |               | The tooltip will be visible near `children` on hover or focus                            |
| `column`    | `boolean`            | `false`       | Column prop set the deriction of the tooltip content                                     |
| `iconLeft`  | `React.ReactNode`    | `null`        | Component to show on the left side of the description (ex.: icon)                        |
| `iconRight` | `React.ReactNode`    | `null`        | Component to show on the right side of the description (ex.: icon or block with actions) |
| `position`  | `string`             | `'top'`       | The positon of the tooltip aginst the relative element                                   |
| `text`      | `React.ReactElement` |               | Information has been described in a tooltip                                              |
| `variant`   | `string`             | `'secondary'` | Variant prop to style Tooltip component                                                  |
