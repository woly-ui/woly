### Variants

Secondary and error variant should be used to focus user attention.

<Playground>
  <ButtonIcon
    icon={<IconSearch />}
    onClick={() => console.info('ButtonIcon clicked')}
    variant="secondary"
  />
  <ButtonIcon
    icon={<IconSearch />}
    onClick={() => console.info('ButtonIcon clicked')}
    variant="secondary"
    filled
  />
  <ButtonIcon
    icon={<IconSearch />}
    onClick={() => console.info('ButtonIcon clicked')}
    variant="danger"
  />
  <ButtonIcon
    icon={<IconSearch />}
    onClick={() => console.info('ButtonIcon clicked')}
    variant="danger"
    filled
  />
</Playground>

### Props

| Name      | Type                                         | Default       | Description                                        |
| --------- | -------------------------------------------- | ------------- | -------------------------------------------------- |
| `icon`    | `React.ReactNode`                            |               | Set the icon component inside the button           |
| `onClick` | `React.MouseEventHandler<HTMLButtonElement>` |               | Callback when button clicked                       |
| `variant` | `string`                                     | `'secondary'` | Variant prop to style ButtonIcon component         |
| `...`     | `HTMLButtonElement`                          | `{}`          | Other props are inherited from `HTMLButtonElement` |
