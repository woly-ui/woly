### prioritys

Secondary and error priority should be used to focus user attention.

<Playground>
  <ButtonIcon
    icon={<IconSearch />}
    onClick={() => console.info('ButtonIcon clicked')}
    priority="secondary"
  />
  <ButtonIcon
    icon={<IconSearch />}
    onClick={() => console.info('ButtonIcon clicked')}
    priority="secondary"
    filled
  />
  <ButtonIcon
    icon={<IconSearch />}
    onClick={() => console.info('ButtonIcon clicked')}
    priority="danger"
  />
  <ButtonIcon
    icon={<IconSearch />}
    onClick={() => console.info('ButtonIcon clicked')}
    priority="danger"
    filled
  />
</Playground>

### Props

| Name       | Type                                         | Default       | Description                                        |
| ---------- | -------------------------------------------- | ------------- | -------------------------------------------------- |
| `icon`     | `React.ReactNode`                            |               | Set the icon component inside the button           |
| `onClick`  | `React.MouseEventHandler<HTMLButtonElement>` |               | Callback when button clicked                       |
| `priority` | `string`                                     | `'secondary'` | Priority prop to style ButtonIcon component        |
| `...`      | `HTMLButtonElement`                          | `{}`          | Other props are inherited from `HTMLButtonElement` |
