`MenuList` is a list of items related to a single object. 
A MenuLis displays a list of choices on a temporary surface. It appears when the user interacts with a button, or other control.
This component can be used in menu dropdown or input autosuggestion components.

Simple menus open over the anchor element by default. 
When close to a screen edge, simple menus vertically realign to make sure that all menu items are completely visible.

MenuItem
This component defines a selectable item in a MenuList. It should only be used as a descendant of a MenuList. 
It does not need to be a direct child of the MenuList component.

### Example

<Playground>
  <MenuList
      menu={[
        { id: 0, text: 'Red' },

### Kinds

| Name       | Description                                               |
| ---------- | --------------------------------------------------------- |
| `MenuList` | Base MenuList. Useful for creating a new kind of MenuList |

### Props

| Name      | Type                     | Default     | Description                              |
| --------- | ------------------------ | ----------- | ---------------------------------------- |
| `menu`    | `Array<React.ReactNode>` |             | An array of menuList items               |
| `variant` | `string`                 | `'default'` | Variant prop to style MenuList component |
