import { MenuList } from './index';

`MenuList` is a list of items related to a single object. This component can be used in menu dropdown or input autosuggestion components.

### Example

<div style={{ maxWidth: '300px' }}>
  <MenuList
    menu={[
      { id: 0, item: 'Red' },
      { id: 1, item: 'Green' },
      { id: 2, item: 'Blue' },
    ]}
  />
</div>

```tsx
const Example = () => (
  <MenuList
    menu={[
      { id: 0, item: 'Red' },
      { id: 1, item: 'Green' },
      { id: 2, item: 'Blue' },
    ]}
  />
);
```

### Kinds

| Name       | Description                                               |
| ---------- | --------------------------------------------------------- |
| `MenuList` | Base MenuList. Useful for creating a new kind of MenuList |

### Props

| Name      | Type                     | Default     | Description                              |
| --------- | ------------------------ | ----------- | ---------------------------------------- |
| `menu`    | `Array<React.ReactNode>` |             | An array of menuList items               |
| `variant` | `string`                 | `'default'` | Variant prop to style MenuList component |
