import { Button } from './index'
import { Playground, block } from 'box-styles'
import { PlusIcon, SearchIcon } from 'icons';

`Button` allows users to take actions and make choices with a single tap.

### Example

<Playground>
  <Button
    icon={<SearchIcon />}
    onClick={() => alert('Hi!')}
    text="Just a button"
    variant="primary"
  />
</Playground>

<!-- <ButtonExample /> -->

### Variants

Primary and secondary variants are should be used to focus user attention.

<Playground>
  <Button text="The main action on the page" variant="primary" />
  <Button text="Alternative action" variant="secondary" />
</Playground>

### Icons

Button can be used with icon on the left side.

<Playground>
  <Button text="Icon on the left" variant="primary" icon={<PlusIcon />} />
  <Button text="Disabled button" variant="primary" icon={<SearchIcon />} disabled />
</Playground>

### Sizes

Size controlled by the `component-level` block property not from the props.

<Playground>
  <block.S>
    <Button variant="primary" text="Small" icon={<PlusIcon />} />
  </block.S>
  <block.M>
    <Button variant="primary" text="Normal" icon={<PlusIcon />} />
  </block.M>
  <block.L>
    <Button variant="primary" text="Large" icon={<PlusIcon />} />
  </block.L>
  <block.XL>
    <Button variant="primary" text="Extra Large" icon={<PlusIcon />} />
  </block.XL>
  <block.H>
    <Button variant="primary" text="Huge" icon={<PlusIcon />} />
  </block.H>
</Playground>

### Container

Buttons can be placed inside the container. Button width is equal to container size.

<Playground direction="horizontal">
  <block.S>
    <Button variant="primary" text="Small" />
  </block.S>
  <block.S>
    <Button variant="primary" text="Small" icon={<PlusIcon />} />
  </block.S>
</Playground>

### Props

| Name      | Type                | Default     | Description                                                |
| --------- | ------------------- | ----------- | ---------------------------------------------------------- |
| `icon`    | `React.ReactNode`   | `null`      | Component to show on the left side of the text (ex.: Icon) |
| `text`    | `React.ReactNode`   |             | Text appeared on the button                                |
| `variant` | `string`            | `'default'` | Variant prop to style Input component                      |
| `...`     | `HTMLButtonElement` | `{}`        | Other props are inherited from `HTMLButtonElement`         |
