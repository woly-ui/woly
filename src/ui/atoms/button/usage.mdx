import {Button} from './index'
import {block, Playground} from 'lib/playground'
import {IconPlus, IconSearch} from 'static/icons';

`Button` allows users to take actions and make choices with a single tap.

### Example

<Playground>
  <Button
    icon={<IconSearch />}
    onClick={() => alert('Hi!')}
    text="Just a button"
    variant="default"
  />
  <Button
    icon={<IconSearch />}
    onClick={() => alert('Hi!')}
    text="Just a button"
    variant="default"
    outlined
  />
</Playground>

### Width

Button can span the entire width of the container or have a fixed width

<Playground direction="horizontal">
  <Button
    icon={<IconSearch />}
    onClick={() => alert('Hi!')}
    text="Just a button"
    variant="primary"
  />
  <Button
    icon={<IconSearch />}
    onClick={() => alert('Hi!')}
    text="Just a button"
    variant="primary"
    fullWidth
    outlined
  />
</Playground>

### Variants

Primary and danger variants are should be used to focus user attention.

<Playground>
  <Button text="The main action on the page" variant="primary" />
  <Button text="The main action on the page" variant="primary" outlined />
  <Button text="Alternative action" variant="danger" outlined />
  <Button text="Alternative action" variant="danger" />
</Playground>

Secondary variant should be used as a default variant.

<Playground>
  <Button text="Alternative action" variant="secondary" />
  <Button text="Alternative action" variant="secondary" outlined />
</Playground>

Tertiary variant should be used as an alternative variant to secondary and can`t be used with outline.

<Playground>
  <Button text="Alternative action" variant="tertiary" />
</Playground>

### Icons

Button can be used with icon on the left side.

<Playground>
  <Button text="Icon on the left" variant="primary" icon={<IconPlus />} />
  <Button text="Disabled button" variant="primary" icon={<IconSearch />} disabled />
</Playground>

### Sizes

Size controlled by the `component-level` block property not from the props.

<Playground>
  <block.N>
    <Button variant="primary" text="None" icon={<IconPlus />} />
  </block.N>
  <block.S>
    <Button variant="primary" text="Small" icon={<IconPlus />} />
  </block.S>
  <block.M>
    <Button variant="primary" text="Normal" icon={<IconPlus />} />
  </block.M>
  <block.L>
    <Button variant="primary" text="Large" icon={<IconPlus />} />
  </block.L>
  <block.XL>
    <Button variant="primary" text="Extra Large" icon={<IconPlus />} />
  </block.XL>
  <block.H>
    <Button variant="primary" text="Huge" icon={<IconPlus />} />
  </block.H>
</Playground>

### Container

Buttons can be placed inside the container. Button width is equal to container size.

<Playground direction="horizontal">
  <block.S>
    <Button variant="primary" text="Small" />
  </block.S>
  <block.S>
    <Button variant="primary" text="Small" icon={<IconPlus />} />
  </block.S>
</Playground>

### Props

| Name      | Type                | Default       | Description                                                |
| --------- | ------------------- | ------------- | ---------------------------------------------------------- |
| `icon`    | `React.ReactNode`   | `null`        | Component to show on the left side of the text (ex.: Icon) |
| `text`    | `React.ReactNode`   |               | Text appeared on the button                                |
| `variant` | `string`            | `'secondary'` | Variant prop to style Input component                      |
| `...`     | `HTMLButtonElement` | `{}`          | Other props are inherited from `HTMLButtonElement`         |
