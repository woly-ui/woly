`notification` display the notification message globaly.

### Example

<NotificationExample />

export const Example = `
<notification.Base message="Your transaction has been completed" />
`.trim();

<HighlightCode code={Example} />

### Kinds

| Name   | Description                                                       |
| ------ | ----------------------------------------------------------------- |
| `Base` | Base notification. Useful for creating a new kind of notificaiton |

### Props

| Name      | Type              | Default | Description                                |
| --------- | ----------------- | ------- | ------------------------------------------ |
| `message` | `React.ReactNode` |         | Message that will be shown in notification |
