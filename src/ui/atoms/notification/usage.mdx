import { NotificationExample } from './example';

`Notification` display the notification message globaly.

### Example

<NotificationExample />

```tsx
const Example = () => (
  <Notification message="Your transaction has been completed" />
);
```

### Kinds

| Name           | Description                                                       |
| -------------- | ----------------------------------------------------------------- |
| `Notification` | Base notification. Useful for creating a new kind of notificaiton |

### Props

| Name      | Type              | Default     | Description                                  |
| --------- | ----------------- | ----------- | -------------------------------------------- |
| `message` | `React.ReactNode` |             | Message that will be shown in notification   |
| `variant` | `string`          | `'default'` | Variant prop to style Notification component |
