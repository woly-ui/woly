`Chip` is used for categorizing or markup. 
Chips allow users to enter information, make selections, filter content, or trigger actions. 
While buttons are expected to appear consistently and with familiar calls to action, chips should appear dynamically as a group of multiple interactive elements.

Chips support content labeling for accessibility and are readable by most screen readers, such as TalkBack. 
Text rendered in chips is automatically provided to accessibility services. Additional content labels are usually unnecessary.

When a chip can be removed a special cancel icon will appear at the far right to indicate this. 
The Remove Icon override provides the means to trigger this functionality.


When using a Chip, the obround will automatically size itself to fit the content and you only need to adjust the size of the instance accordingly. 
If the Chip is used as an action trigger for related content avoid using it in scenarios where there is only one available action, 
or for configurations presenting only one selectable option.

As a standalone component, the most common use will be in some form of input, so some of the behaviour demonstrated here is not shown in context.

### Clickable

Chip can be a sipmple component or can be clickable component. 

In case when chip is clickable, it defined change appearance on focus, hover, and click.

If chip consists another clickable component (eg button), this property defined will display a component which changes appearance on hover.

<Playground>
  <Chip variant="default" onClick={() => console.info('click')}>
    Cklicked Chip
  </Chip>
  <Chip
    variant="default"
    action={<CloseIcon />}
    onActionClick={() => console.info('ActionIcon clicked')}
  >
    Cklicked Icon
  </Chip>
  <Chip
    variant="default"
    onClick={() => console.info('click')}
    action={<CloseIcon />}
    onActionClick={() => console.info('ActionIcon clicked')}
    disabled
  >
    Disabled Chip
  </Chip>
</Playground>

### Kinds

| Name   | Description                              |
| ------ | ---------------------------------------- |
| `Chip` | Base chip, useful for creating new chips |

### Props

| Name            | Type                                         | Default     | Description                                        |
| ----------------| -------------------------------------------- | ----------- | -------------------------------------------------- |
| `children`      | `string`                                     |             | Text inside chip                                   |
| `action`        | `React.ReactNode`                            | `null`      | Icon to show on the right side of the text         |
| `onActionClick` | `React.MouseEvent<HTMLButtonElement>`        | `null`      | Callback when chip is clicked                      |
| `onClick`       | `React.MouseEvent<HTMLDivElement>`           | `null`      | Callback when chip is clicked                      |
| `variant`       | `string`                                     | `'default'` | Variant prop to style Chip component               |
| `...`           | `HTMLButtonElement`                          | `{}`        | Other props are inherited from `HTMLButtonElement` |
