`Chip` is used for categorizing or markup. Chips can be used inside input or as tags.

### Example

<ChipExample />

```tsx
const Example = () => (
  <Chip
    icon={<CloseIcon />}
    onClick={(event) => console.info('Chip clicked')}
    text="Chip"
    variant="primary"
  />
);
```

### Kinds

| Name   | Description                              |
| ------ | ---------------------------------------- |
| `Chip` | Base chip, useful for creating new chips |

### Props

| Name      | Type                                         | Default     | Description                                        |
| --------- | -------------------------------------------- | ----------- | -------------------------------------------------- |
| `text`    | `React.ReactNode`                            |             | Text inside chip                                   |
| `icon`    | `React.ReactNode`                            | `null`      | Icon to show on the right side of the text         |
| `onClick` | `React.MouseEventHandler<HTMLButtonElement>` | `null`      | Callback when chip is clicked                      |
| `variant` | `string`                                     | `'default'` | Variant prop to style Chip component               |
| `...`     | `HTMLButtonElement`                          | `{}`        | Other props are inherited from `HTMLButtonElement` |
