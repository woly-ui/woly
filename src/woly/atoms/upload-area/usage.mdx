`UploadArea` component is a layer that can be used by developer/designer to create some custom upload component.

Uploading is the process of publishing information (web pages, text, pictures, video, etc.) to a remote server via a web page or upload tool.

- when you need to upload one or more files.

- when you need to show the process of uploading.

- when you need to upload files by dragging and dropping.

As usuall the upolad component consists some content, like upload button or input.
In classic mode file selection dialog pops up when upload button is clicked.

As a successfull result on the screen should be displayed upload data or files.

Also `UploadArea` component should be reused in creation other components, eg drag&drop component.

### Example

  <Text type="L">Empty UploadArea component&nbsp;</Text>
  <div style={{ width: '600px', height: '600px' }}>
    <UploadArea priority="primary" />
  </div>
</Playground>

<Playground direction="vertical">
  <Text type="L">UploadArea component with content&nbsp;</Text>
  <div style={{ width: '600px', height: '600px' }}>
    <UploadArea center priority="primary" content={<Button text="Upload button" />} />
  </div>
</Playground>

<Playground direction="vertical">
  <Text type="L">Scrolled UploadArea component with huge data&nbsp;</Text>
  <div style={{ width: '1200px', height: '600px' }}>
    <UploadArea
      priority="primary"
      content={
        <Table columns={tableHead.length} priority="primary">
          <Thead>
            <Tr>
              {tableHead.map(({ id, name }) => (
                <Th key={id}>{name}</Th>
              ))}
            </Tr>
          </Thead>
          <Tbody>
            {tableRows.map((row) => (
              <Tr>
                {tableHead.map(({ id }) => (
                  <Td key={id}>{row[id]}</Td>
                ))}
              </Tr>
            ))}
          </Tbody>
        </Table>
      }
    />
  </div>
</Playground>

### Props

| Name       | Type              | Default     | Description                                   |
| ---------- | ----------------- | ----------- | --------------------------------------------- |
| `center`   | `boolean`         | `false`     | Whether the component has benn centred or not |
| `content`  | `React.ReactNode` |             | Pass any content within UploadArea component. |
| `priority` | `string`          | `'default'` | Priority prop to style UploadArea component   |
