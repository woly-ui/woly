import {block, FormBlock, Playground, State} from 'lib/playground'
import {Button, HeaderPanel, Input, Label, Surface} from 'ui'

`HeaderPanel` is a navigation component that displays information and actions relating to the current screen.

### Example

<Playground>
  <div style={{ marginBottom: '10px' }}>
    <HeaderPanel>Hello! It`s a HeaderPanel</HeaderPanel>
  </div>
</Playground>

### Props

| Name       | Type              | Default       | Description                                                           |
| ---------- | ----------------- | ------------- | --------------------------------------------------------------------- |
| `children` | `React.ReactNode` |               | Pass any content as a children property within HeaderPanel component. |
| `priority` | `string`          | `'secondary'` | Priority prop to style HeaderPanel component                          |
